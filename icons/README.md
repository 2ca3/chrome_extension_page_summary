# アイコンについて

このディレクトリには以下のサイズのアイコンファイルが必要です：

- icon16.png (16x16 ピクセル)
- icon48.png (48x48 ピクセル)
- icon128.png (128x128 ピクセル)

## アイコンの作成方法

1. オンラインアイコンジェネレーター（例：https://www.favicon-generator.org/）を使用
2. デザインツール（Figma、Canvaなど）で作成
3. 既存の画像から変換

## 推奨デザイン

- シンプルで認識しやすいデザイン
- 背景色は透明または単色
- ページ要約をイメージしたアイコン（例：文書アイコン、要約記号など）

## 暫定対応

開発中は、シンプルな色付き正方形や、以下のコマンドでImageMagickを使用してプレースホルダーを作成できます：

```bash
convert -size 16x16 xc:#4CAF50 icon16.png
convert -size 48x48 xc:#4CAF50 icon48.png
convert -size 128x128 xc:#4CAF50 icon128.png
```

注：Chrome拡張機能をテストする際は、実際のアイコンファイルが必要です。
